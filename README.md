[LedCube for Arduino]


Использование
-------------
1. Откройте файл LedCube-PM-Kirill.ino в Arduino IDE.

2. По желанию добавьте свои эффекты.

3. Загрузите код на Arduino, как описано выше.

4. Наблюдайте за эффектами на своем LedCube!
